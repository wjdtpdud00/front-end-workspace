br 요소는 줄을 바꿀 때 사용한다. 그리고 공백은 한개의 공백만을 표시하기 때문에

 pre 태그는 시작태그에서부터 종료태그까지 
 입력된 내용 그대로 표현하는 태그
 줄바꿈이라던가 여러개의 공백도 바로 인식함

그 밖에 글자를 다루는 태그들

b : 글자를 굵게 표시하는 태그
strong : 글자를 굵게 표시하는 태그

>i : 글자를 기율여서 표시하는 태그, 또는 이모티콘 사용
em : 글자를 기울여서 표시하는 태그

u : 글자에 밑줄이 그어지는 태그
s : 글자에 취소선을 넣어주는 태그 mark : 글자에 형광펜 효과를 넣어주는 태그


small : 글자를 작게 표현하는 태그
기본 글자에 sub : 아래첨자를 나타내는 태그와 sup : 윗첨자를 나타내는 태그
